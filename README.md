# Retail Sales Analysis

## Objective
The main objective of this analysis is to identify the key revenue drivers and how pricing,volume and seasonal changes affects the overall revenue.

## Key Insights
1. Electronics - Stable Category
- Generates Highest Revenue out of all three categories
- Second Highest number of units sold
- Second Highest Average Price per Unit
- Generates Second Highest Revenue per Customer

2. Clothing - Volume-Driven Category
- Highest Number of Units Sold
- Generates Moderate Revenue
- Least Average Price Per Unit
- Generates Least Revenue per Customer

3. Beauty - Premium Niche
- Generate Least Revenue out of all three categories
- Generates Highest Revenue per Customer
- Highest Average Price per Unit
- Sells least number of units

## Conclusion
From the analysis, we can conclude that Electronics Category is the most stable category, Clothing is volume-driven category and Beauty Category is premium niche category.
